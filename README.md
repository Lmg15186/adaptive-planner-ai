# adaptive-planner-ai

Adaptive Planner AI

Smart task planning system developed in Java.
This project uses rule-based logic to organise tasks based on priority, deadline and user energy level.

Features

Add tasks

Define priority (High, Medium, Low)

Define available hours per day

Suggest optimal task order

Energy-based scheduling logic

Purpose

To explore productivity optimisation using programming logic and user behaviour patterns.

Status

In progress – Version 1 (console application)
