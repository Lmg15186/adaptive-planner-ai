# Adaptive Planner AI (Java)

Smart task planner with rule-based logic to suggest an optimal schedule based on:
- Priority (High/Medium/Low)
- Available hours
- User energy level (1–5)

## Features (v1)
- Add tasks (name, priority, hours)
- Sort by priority
- Energy-based suggestion (low energy avoids heavy tasks)

## How to run
Compile:
javac Task.java Planner.java

Run:
java Planner
